## 🎨 Customization Quick Reference

### Change Colors
Edit CSS variables in `<style>` section:
```css
:root {
    --cyan: #00d4ff;        /* Change primary accent */
    --purple: #6c63ff;      /* Change secondary accent */
    --bg-primary: #0a0e1a;  /* Change background */
}
```

### Modify Content
All content is in plain HTML - just find and edit:
- **Header text**: Search for `<h1>`, `<h2>`, etc.
- **Card content**: Look for `<div class="card">`
- **Chat messages**: Find `<div class="chat-message">`

### Add More Scenarios
Duplicate the chat scenario structure:
```html
<div class="chat-messages" id="scenario-3">
    <!-- Add your messages here -->
</div>
```

## ❓ Troubleshooting

### Problem: Animations not working
**Solution**: Ensure you're using a modern browser (Chrome 90+, Firefox 88+, Safari 14+)

### Problem: Glassmorphism effects missing
**Solution**: Backdrop-filter requires recent browser versions. Update your browser.

### Problem: Layout looks broken on mobile
**Solution**: Check viewport meta tag is present (it is by default)

### Problem: Google Fonts not loading
**Solution**: Check internet connection - fonts load from CDN

### Problem: Smooth scroll not working
**Solution**: Some older browsers don't support smooth scroll behavior
