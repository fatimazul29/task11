## 📊 Challenge Requirements Summary

### **What Participants Must Build:**
- AI-powered Hospital Appointment Support Agent
- Platform: **n8n** (Workflow Automation)
- Integrations: OpenAI/Gemini, Google Sheets/Notion, Webhook/Chat Interface

### **Agent Capabilities:**
- Answer hospital service & doctor queries
- Book appointments automatically
- Respond ONLY from provided dataset
- Maintain professional & polite tone
- Prevent misuse and data manipulation

### **Dataset Fields Required:**
- Doctor Name
- Specialization
- Available Days
- Consultation Fee
- Hospital Department
- Services Provided

### **Key Rules:**
1. **Data-Restricted**: No information outside the dataset
2. **Professional Tone**: Formal, professional, polite
3. **Misuse Protection**: Deflect inappropriate requests to reception
