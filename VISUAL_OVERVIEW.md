## 🎬 Animation Timeline

```
PAGE LOAD
    │
    ├─ 0.0s: Header Fade In Down (1s duration)
    ├─ 0.3s: Skills Bar Fade In (1s duration)
    ├─ 1.0s: First Section Reveal (on scroll)
    │
SCROLL TRIGGER
    │
    ├─ Section enters viewport
    │   └─ Fade In + Translate Up (0.8s)
    │
HOVER EFFECTS
    │
    ├─ Cards: Lift (-5px) + Glow Shadow
    ├─ Buttons: Lift (-3px) + Enhanced Glow
    ├─ Skill Tags: Lift (-3px) + Brighter BG
    └─ Table Rows: Background Highlight
    │
CONTINUOUS
    │
    ├─ Hospital Icon: Pulse Scale (2s loop)
    ├─ CTA Button: Pulse Glow (2s loop)
    ├─ Flow Arrows: Opacity Flow (2s loop)
    ├─ Typing Dots: Bounce (1.4s loop, staggered)
    ├─ Chat Widget: Pulse (2s loop)
    └─ Gradient Text: Background Shift (3s loop)
```
