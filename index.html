Overview

Participants are required to build an AI-powered Hospital Appointment Support Agent using the n8n automation platform.

The purpose of this agent is to assist patients by:

  • Answering hospital services and doctor-related inquiries
  • Helping them book appointments efficiently
  • Providing instant, accurate information from a verified dataset

The AI agent must strictly follow the rules and respond only based on the provided hospital dataset.

5

Functional Requirements

5.1

🔒 Data-Restricted Responses

The AI agent must:

  • ✅ Provide answers only from the provided dataset
  • ✅ Not generate or assume information beyond the dataset
  • ✅ Never fabricate doctor names, fees, or availability

If information is not available, respond politely:

"I apologize, but I can only provide information based on the available hospital data. For further details, please contact our hospital reception."
5.2

🎩 Professional Tone

All responses must be:

  • ✅ Formal — Using proper language and grammar
  • ✅ Professional — Maintaining healthcare industry standards
  • ✅ Polite — Courteous and empathetic communication
5.3

🛡️ Misuse Protection

If a user tries to:

  • ❌ Ask unrelated questions (e.g., weather, recipes, jokes)
  • ❌ Modify doctor or service information
  • ❌ Perform actions outside the agent's scope
  • ❌ Misuse the chatbot (spam, offensive language)

The agent must respond:

"For further assistance, please contact our hospital reception at: [Phone Number]"
